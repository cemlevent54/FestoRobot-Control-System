Detailed Description

Manages the robot's components and interactions.

The Robot class initializes and integrates different components of the robot, including sensors, navigation, and mapping systems. It serves as the central point for coordinating robot functionalities.

Robot::Robot ( )

Default constructor for the Robot class.

Initializes all the robot's components and subsystems, setting up the necessary interfaces for robot operation.

This constructor initializes the Robot object by setting the controller to NULL and creating instances of BumperSensor, ForceSensor, and SonarSensor.
