Detailed Description

Represents a grid-based map system for managing map data.

The MAP class allows for the insertion of points into the map, accessing and modifying the grid values, clearing the map, and displaying the map's information. The map is stored in a 2D grid, and each cell can hold integer values representing the map's structure.
