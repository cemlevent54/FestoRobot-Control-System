Detailed Description

Handles file reading and writing operations.

The Record class provides methods to open, close, read, and write lines to a file. It also includes methods to set the file name, read a line, and write a line to the file. Additionally, the class overloads the stream insertion (<<) and extraction (>>) operators for convenient input and output handling of file data.
