Detailed Description

Provides functionality to interact with an infrared (IR) sensor.

The IRSensor class reads and updates distance measurements from the IR sensor. It provides utility functions to access sensor values, print sensor data, and retrieve the sensor type. The class integrates with the robotic system via the FestoRobotSensorInterface.
