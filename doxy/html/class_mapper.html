Detailed Description

Maps the environment using lidar sensor data and robot control data.

The Mapper class is responsible for managing and updating a grid-based map using the LidarSensor to detect obstacles and the RobotControler to track the robot's position and movements. It offers functionality to update the map, record it to a file, and visually display the current state of the map.
